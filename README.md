# RadiantLuma

**RadiantLuma** is a custom fork of [Luma3DS](https://github.com/LumaTeam/Luma3DS) that adds more features for controlling your Nintendo 3DS screen brightness.

Inspired by [Starlight3DS](https://github.com/Billyfury31/Starlight3DS).

---

## Features

### Brightness Boost
Significantly increase your screen brightness **beyond the standard 3DS limits** for improved visibility in bright environments.

- Temporary brightness pushes the screen luminance above the default 172
- Works on both top and bottom screens
- Useful for outdoor use and also for removing the infamous "yellow screen tint" (tested with a New 3DS plugged in while idle with a white background, significantly clearing the yellow tint).

### Permanent Brightness Recalibration
Customize the five default brightness levels to your exact preferences using Nutez's Permanent Brightness Recalibration feature.

- Modify any of the 5 brightness preset levels
- Values range from 0 to 172
- Changes are written to NAND and persist across reboots

### Night Light Mode
A blue light filter with automatic scheduling, perfect for late-night sessions.

- **Customizable schedule**: Set specific times for Night Light to turn on/off
- **Blue light reduction**: Easier on your eyes during nighttime use. Temperature value can be customized
- **Screen dimming**: Set to any brightness level (1-5)
- **Auto-toggle**: Activates automatically based on your configured time
- **Sleep mode support**
